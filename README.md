# PocketAI

PocketAI is a smart expense tracking and AI assistant application built with Flutter. It helps users manage their finances through intuitive visualization, receipt scanning, and voice-activated AI assistance.

> [!NOTE]
> This project was developed as part of the **Finovia 2025 Hackathon**.

## Features

- **Authentication**: Secure login and sign-up functionality for user accounts.
- **Dashboard**: View recent expenses and visualize spending habits with interactive charts.
- **Receipt Scanning**: Capture receipts using the camera and automatically extract transaction details using OCR.
- **AI Chatbot**: Interact with an intelligent assistant via text or voice to query financial data or get advice.
- **Voice Interaction**: Record voice commands to interact with the chatbot hands-free.

## Tech Stack

- **Framework**: [Flutter](https://flutter.dev/)
- **Language**: [Dart](https://dart.dev/)
- **Key Packages**:
  - [`dio`](https://pub.dev/packages/dio): For handling network requests.
  - [`record`](https://pub.dev/packages/record): For audio recording capabilities.
  - [`image_picker`](https://pub.dev/packages/image_picker): For capturing images from the camera or gallery.
  - [`fl_chart`](https://pub.dev/packages/fl_chart): For rendering graphical charts.
  - [`google_fonts`](https://pub.dev/packages/google_fonts): For using custom fonts.
